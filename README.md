# 🔐 Password Strength Checker

A simple Python tool to evaluate password strength.  
It checks for 5 key components of strong passwords:

- ✅ Minimum length (8 characters)
- ✅ Uppercase letters
- ✅ Lowercase letters
- ✅ Numbers
- ✅ Special characters

## 🚀 How to Use

Run the script:
```bash
python checker.py
